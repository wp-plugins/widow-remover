Removes widows from any content in the content editor or title by adding non-breaking spaces where necessary.

== Description ==
Widow Remover uses non-breaking spaces to ensure your content added via the content editor will not allow a single word on the last line of your content. Works an all basic tags utilized by the WordPress editor.

The plugin features an option to to wrap affected content in start and end comments as well as an option to use the plugin on fields added via the Advanced Custom Fields plugin.

The plugin works out of the box. No setup or configuration is necessary. Simply activate the plugin and it will start working immediately.

== Installation ==
Upload \'widow-remover\' to the \'/wp-content/plugins/\' directory

Activate the plugin through the \'Plugins\' menu in WordPress

The plugin is up and running properly with no additional steps

Note: If you are using a caching plugin you may have to clear the cache in order for the plugin to start working.
